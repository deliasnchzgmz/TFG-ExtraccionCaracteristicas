(ESPAÑOL)
Para el correcto funcionamiento del proyecto es necesario contar con la carpeta 'resources' que se incluye con el programa.
En esta carpeta debe haber a su vez tres elementos:
	- out: contiene los archivos que se generan y modifican con el funcionamiento del programa
	- sheets: contiene las imágenes disponibles para ejecutar el programa. Se pueden añadir imágenes a esta ubicación.
	- templates: contiene las imágenes plantilla para la detección de los elementos en la imagen. NO MODIFICAR ESTA CARPETA.
	- logo_uc3m.ico: icono para el programa.
	

(ENGLISH)
In order to guarantee the performance of this program, it is compulsory to keep the 'resources' folder in the same directory.
This folder must contain the following items:
	- out: it contains the files which are generated and modified while the program is running
	- sheets: it includes the available images for the program to use. It is possible add new images if desired.
	- templates: it contains the template images used to detect elements. DO NOT MODIFY THIS FOLDER.
	- loco_uc3m.ico: programs icon.
